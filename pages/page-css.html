LE CSS

C'est quoi le CSS ?

Définition

Cascading Style Sheets (CSS) est un langage de feuille de style utilisé pour définir l'apparence et le formatage des documents web. CSS contrôle la mise en page, les couleurs, les polices, et d'autres aspects visuels des éléments HTML.

En plus simple :

Les LEGO

Imagine que tu as une boîte de LEGO. Les pièces de LEGO sont comme les éléments HTML de ta page web, et CSS, c'est comme les autocollants et la peinture que tu utilises pour rendre ta création encore plus jolie et colorée.

Comment ça marche ?

Pense à HTML comme le squelette de ta page web. C'est ce qui tient tout en place. CSS, c'est comme les vêtements et les accessoires que tu mets sur ce squelette pour le rendre beau.

Caractéristiques principales du CSS

  1. Séparation du contenu et de la présentation :
    • HTML : Utilisé pour structurer le contenu d'une page web.
    • CSS : Utilisé pour styliser et positionner le contenu.
  2. Cascade et héritage :
    • Cascade : Les styles peuvent se cumuler et se chevaucher. Les styles définis plus bas dans le fichier peuvent écraser les styles définis plus haut.
    • Héritage : Certains styles appliqués à un élément parent peuvent être hérités par ses éléments enfants.
  3. Sélecteurs :

    Types de sélecteurs : CSS utilise différents types de sélecteurs pour cibler les éléments HTML à styliser, comme les sélecteurs de type, de classe, d'ID, d'attribut, et les pseudo-sélecteurs :

    • Sélecteur de type : p sélectionne tous les p (paragraphe).
    • Sélecteur de classe : .classe sélectionne tous les éléments avec la classe "class".
    • Sélecteur d'ID : #id sélectionne l'élément avec l'ID "id".
  4. Propriétés et valeurs :
    • Les propriétés définissent les aspects visuels des éléments, comme color, font-size, margin, padding, border, etc.
    • Les valeurs sont les paramètres appliqués aux propriétés.
