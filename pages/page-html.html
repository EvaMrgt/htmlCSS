C'est quoi le HTML ?

Rien de plus simple :

Définition

HyperText Markup Language (HTML) est un langage de balisage utilisé principalement pour créer et organiser le contenu sur le web. Il permet aux développeurs de structurer des textes, des images, des liens, des tableaux, des listes, des formulaires et d'autres types de contenu multimédia.

Le développeur s'occupe donc en réalité de toute la réalisation des fonctionnalités techniques d'un site internet.

Alors oui, mais en français, ça donne quoi ?

Les LEGO

Imagine que créer une page web, c'est un peu comme construire une maison en LEGO. Les briques LEGO sont les différentes parties de ta maison, comme les murs, le toit, et les fenêtres. HTML, c'est comme les instructions qui te disent comment assembler toutes ces briques pour créer ta maison.

OK. J'ai compris, mais comment mettre ça en pratique ?

Les bases

  1. Balises : Les balises sont comme les briques LEGO. Elles disent à ton navigateur (comme Chrome ou Firefox) comment afficher chaque partie de ta page. Les balises sont entourées de chevrons ">". Par exemple :
    • h1 : C'est une grosse brique pour un grand titre, comme le titre d'un livre
    • p : C'est une brique pour un paragraphe, comme une phrase ou un bloc de texte.
  2. Structure : Une page HTML a une structure bien précise, un peu comme les fondations d'une maison. Voici comment elle est organisée :
    • DOCTYPE html : C'est comme dire "Ceci est une maison LEGO".
    • html : C'est comme la boîte qui contient toutes tes briques.
    • head : C'est l'endroit où tu mets des choses importantes pour la maison, comme les plans ou les informations sur les pièces.
    • body : : C'est le corps de la maison, où tu mets toutes les pièces que les gens peuvent voir, comme les chambres, le salon, etc.

Conclusion

Créer une page web avec HTML, c'est comme assembler une maison avec des briques LEGO en suivant des instructions. Chaque balise est une brique qui aide à construire et organiser ta page pour que tout le monde puisse la voir et l'utiliser. Voilà, tu es maintenant prêt à commencer à construire tes propres pages web !
